# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - 2026-05-01

### Added
- Initial release of the tool.
- Recursive scanning of directories to find `.aider.chat.history.md` files.
- Aggregation of costs and token usage by project and by day.
- Rich terminal display for tables and summaries.
- CI integration with Ruff and strict Mypy.
